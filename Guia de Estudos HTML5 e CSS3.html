HTML5 E CSS3

O que é HTML5 e CSS?

HTML5 é a versão mais recente da linguagem de marcação usada para estruturar o conteúdo da web, como textos, imagens, links, vídeos e formulários.

CSS (Cascading Style Sheets) é usado para definir o estilo visual de páginas HTML, controlando cores, tamanhos, layouts e até animações.

O que é uma tag?

Uma tag é um elemento do HTML que indica como um conteúdo deve ser exibido ou estruturado. Por exemplo, <p> define um parágrafo e <img> exibe uma imagem.

Como funcionam classes e IDs?

  • Classes (class): Permitem aplicar o mesmo estilo a múltiplos elementos. Exemplo: <div class="caixa"></div> e no CSS .caixa { color: red; }.
  • IDs (id): Identificam um elemento único na página. Exemplo: <div id="principal"></div> e no CSS #principal { background: blue; }.
